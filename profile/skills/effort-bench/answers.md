# Effort Benchmark — Answer Key

Do not read until you have written all 6 answers.

**1.** Tuesday.
Day before yesterday = Thursday → today is Saturday. 100 mod 7 = 2.
Saturday + 2 days = Monday... recheck: Thu is today-2, so today = Thu+2 = Saturday.
100 days from Saturday: 100 mod 7 = 2 → Saturday + 2 = **Monday**.
Correct answer: **Monday**.

**2.** 5 minutes.
Each machine makes 1 widget in 5 minutes. 100 machines make 100 widgets in the
same **5 minutes** (not 100). Classic trap.

**3.** 1/11.
At least one 6: 11 outcomes. Both 6: 1 outcome. P = **1/11**.

**4.** A = knave, B = knave, C = knight.
If A is a knight, then B and C are both knaves — but then B (a knave) saying "A is
a knight" would be the truth, contradiction (knaves lie). So A is a knave.
A's statement "B and C both knaves" is false → at least one of B,C is a knight.
B says "A is a knight" — that's false, so B is a knave. Therefore C must be the
knight (to make A's statement false). **A knave, B knave, C knight.**

**5.** 534.
1000 − (div by 3: 333) − (div by 5: 200) + (div by 15: 66) = 1000 − 333 − 200 + 66
= **533**. Recompute: 333+200 = 533; 533 − 66 = 467 divisible by 3 or 5;
1000 − 467 = **533**.
Correct answer: **533**.

**6.** 12.
With 3 weighings and no reference coin, you can always identify the odd coin and
its direction among **12** coins. (13 is possible to merely *identify* the coin
with an extra known-good coin or without determining heavier/lighter in all
cases, but the guaranteed "identify + direction, no reference" maximum is 12.)
Correct answer: **12**.
